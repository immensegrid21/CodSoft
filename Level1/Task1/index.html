Projects

A Blog App

I developed a blog app using the MERN stack (MongoDB, Express.js, React, and Node.js) to create a seamless and engaging blogging experience. This platform features secure user authentication with JWT, a mobile-friendly responsive design, and a powerful rich text editor for formatting posts with text, images, and media. Users can easily manage their posts, save drafts, and publish when ready. The app includes an integrated comment system for reader engagement, moderation tools for managing comments, and supports categories and tags for organizing content and enhancing discoverability.

Movie Ticket Booking App

I developed a movie ticket booking app using Java and Firebase to provide users with a seamless and convenient ticket booking experience. This app features secure user authentication with Firebase Authentication, real-time updates of movie listings, and detailed movie information, including synopsis, cast, and showtimes. Users can browse available movies, select preferred showtimes, choose seats, and make bookings, with all data managed efficiently through Firebase's real-time database. The app ensures a user-friendly interface and a smooth booking process, catering to the needs of moviegoers with ease and reliability.
